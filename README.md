# 🎬 Netflix Content Intelligence Dashboard

## 📌 Project Overview

The **Netflix Content Intelligence Dashboard** is a data analytics
project that explores and visualizes Netflix's movie and TV show
catalog. The project performs **data cleaning, exploratory data analysis
(EDA), and interactive visualization** to understand patterns in Netflix
content.

Using **Python, Pandas, and Streamlit**, the project converts raw
Netflix data into meaningful insights and an interactive dashboard.

------------------------------------------------------------------------

## 🎯 Objectives

-   Analyze Netflix's catalog of Movies and TV Shows
-   Identify trends in content released over the years
-   Explore distribution of content across countries
-   Analyze ratings and genres of Netflix titles
-   Build an interactive dashboard for visual exploration

## 📊 Exploratory Data Analysis (EDA)

The dataset was analyzed using **Jupyter Notebook** to understand
patterns and trends.

Key analysis performed: - Dataset inspection and cleaning - Handling
missing values - Distribution of Movies vs TV Shows - Content release
trends by year - Country-wise content production - Ratings and genre
analysis

Libraries used: - Pandas - NumPy - Matplotlib - Seaborn

------------------------------------------------------------------------

## 📈 Dashboard Features

The project includes an **interactive Streamlit dashboard** that allows
users to explore Netflix content visually.

Dashboard highlights: - Total Movies and TV Shows count - Content
distribution by type - Release trend over the years - Top countries
producing Netflix content - Ratings distribution - Interactive filters
for exploring data
